MPF Installation
================

Installing MPF is easy, and there are a few ways you can do it. (Note
this page is only about the actual installation of the MPF software
and hardware drivers. You can use our :doc:`step-by-step tutorial </tutorial/index>`
to walk you through actually building your game config with MPF.)

OS Platforms supported
----------------------

MPF works on several different platforms, on both Intel x86 and ARM processors,
and in 64-bit and 32-bit systems.

Due to some peculiarities of Python and graphics libraries, only certain versions
of Python are supported on certain OSes:

=============================================== ==============
OS                                              Python version
=============================================== ==============
Windows 7, Windows 8, Windows 10 (32-bit)       3.4
Windows 7, Windows 8, Windows 10 (x64 / 64-bit) 3.4
Mac OS X (Yosemite or El Capitan)               3.5
Linux (on both Intel and ARM hardware)          3.4 or 3.5
=============================================== ==============
